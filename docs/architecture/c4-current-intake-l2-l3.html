Level 3 — Component View (inside Apps Script Web API)

doPost(e) — Controller
Entry point. Parses JSON/form, handles CORS, returns JSON response.
validateLead(payload)
Required fields, email format, budget/timeline sanity checks.
spamGuard(payload, e)
Honeypot check, basic rate-limit key (IP/user agent/timestamp).
normalizeLead(payload)
Canonical field mapping + trim + timestamp + source metadata.
writeLeadToSheet(lead)
Append row to Leads sheet, set initial status new.
sendLeadEmail(lead)
Formats plaintext/HTML summary and sends to CHC inbox.
buildSuccessResponse(id)
Returns success token/lead id for thank-you UX.
logError(err, context)
Write diagnostics to script logs + optional Errors sheet.
